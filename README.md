# Real-Time GPS Tracker

#### Description
The project explores the fundamentals of geolocation APIs in JavaScript. The project makes use of MapBox API as a resource for it to function.

#### Features
- Shows user location 
- Displays marker 
- Uses MapBox 
- Professional Design

#### Usage
Click a button to show current location.

#### Technologies / Libraries Used
- MapBox GL JS
- Geolocation API
- HTML
- CSS 
- JavaScript
